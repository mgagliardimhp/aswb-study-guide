# ASWB Clinical Exam Hub v1

Starter Vite + React scaffold for an ASWB Clinical exam study hub.

## Included in this version

- React Router
- Study Map
- Learn Mode
- Practice Mode
- Starter JSON data for all 4 Clinical domains and subdomains
- Starter KSA data
- One fuller Learn Mode slice for Domain II / IIB
- Sample practice questions

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Starter routes

- `/` Home
- `/study-map`
- `/domain/:domainId`
- `/learn/:ksaId`
- `/practice`
- `/practice/session`
